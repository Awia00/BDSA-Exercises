class  InheritanceRPC
 The class takes a string as input, and if the string is in the format of a reverse Polish calculator expression, then it will calculate and return the result. Features Can handle the operators: +, -, *, /, sqrt, cos, sin, abs and pow (^) Can handle negative numbers if written in the form -x Can handle doubles if written in the form x,x Tokens are separated by a whitespace. If the input is invalid an exception is thrown. The class uses a dictionary<string, IOperation> to handle the operations. To add an operation simply add it to the dictionary.
