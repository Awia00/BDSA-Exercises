Generates a regular expression string based on the user input. Three different commands are available, but there can only be one command pr. regular expression. Commands\+: + between two words, $\ast$ prefix, $\ast$ suffix. 


\begin{DoxyItemize}
\item means that two adjacent keywords should be present one after the other, e.\+g. text + file matches “text file” but does not match “text big file”
\item after a keyword means search for words starting with keyword e.\+g. inter$\ast$ matches “internet”
\item before a keyword means search for words ending with keyword e.\+g. $\ast$net matches “internet” and “intranet” 
\end{DoxyItemize}
